# AGENTS.md

## About this repo

Legacy classic (1.x) codebase for Marvin's Awesome iRacing App. No longer updated upstream — the active app is [MAIRA Refactored (2.x)](https://github.com/mherbold/MarvinsAIRARefactored). This working copy is used for local documentation and analysis of the legacy code (see `docs/`), not active feature development.

## Git Workflow

- `origin` (`elderfo/MarvinsAIRA`) is our own fork — pushing directly to it is fine.
- **Never commit or open a PR against the upstream repo we forked from (`mherbold/MarvinsAIRA`).** That's the original source repo; nothing gets contributed back to it.
- Commit directly to `main` and push to `origin`. No branches or PRs are needed for work in this repo.
